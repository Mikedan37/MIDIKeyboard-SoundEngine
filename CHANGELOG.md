# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Velocity-sensitive keyboard matrix implementation
- MSQT32 shift register driver
- 2-phase scanning for velocity detection
- Comprehensive test suite (no hardware required)
- macOS menu bar synthesizer GUI
- Real-time polyphonic audio synthesis
- USB HID + MIDI dual mode support

### Changed
- Migrated from direct GPIO to matrix-based scanning
- Updated to velocity-sensitive key detection
- Improved test coverage and validation

### Fixed
- Various compilation errors and warnings
- Function name mismatches
- Missing includes and declarations
- Test suite compilation issues

## [1.0.0] - 2024

### Added
- Initial release
- QWERTY keyboard mode
- MIDI controller mode
- Python synthesizer engine
- macOS integration
- Pico firmware with TinyUSB
- Documentation suite
